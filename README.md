simple_calculator. простой калькулятор из двух чисел и действий: сложение, вычитание, умножения и деления. изначально был поставлен на if else, но сразу же был пепеписан на более быстрый и приятно читабельный switch case

calculator_With_history. тот же простой калькулятор из двух чисел, но уже из большего количества действий: сложение, вычитание, умножения, деления, выход, история. сделал сам код в один цикл для того чтобы калькулятор работал вечно пока не напишешь действие выход. сделано так было для обучения вектору.
